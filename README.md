# Task-pro-Manager


MCA Final Semester Internship Project

## Description
Task Pro Manager is a web-based task management system designed to manage
academic and internship-related tasks using role-based access control.

## Tech Stack
- Frontend: React.js
- Backend: FastAPI (Python)
- Database: Microsoft SQL Server

## User Roles
- System Administrator
- Project Supervisor
- Task Executor (Student)

## Features
- User Authentication
- Role-Based Access Control
- Project Management
- Task Assignment & Tracking
- Dashboard & Reports
